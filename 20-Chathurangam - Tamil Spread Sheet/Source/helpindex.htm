                                                                                                File

New : New Files can be opened through this menu. The current will be visible. this
         operation can be done through by clicking new Icon.

Open : Already saved files can be opened through this menu. Icon is available for this     option.

Close: Currently opened files can be closed. All sheets will be closed.

Save : The file with untitled.txls cannot be saved. The sheet with some data can be
          saved in .txls file format. Icon option is also provided for this.

SaveAs : To get the copy of the file one can use this option. Also the file with some
             modification can be saved in different file name.

Print :  Sheets with current data can be printed. Icon provided for this too.

Properties : File properties can be seen through this menu item.

RecentFiles : Recently viewed files list are available (maximum of 5 files) here.

Exit :  It will close the application by clicking this.

                                                                                             Tools

Cut : The data from a single cell or multiple cells can be cut and pasted somewhere.
         The data will not be there in source.

Copy : The data from a single cell or multiple cells can be copied from source to
            somewhere. The data in the source will be retained.

Paste :The data content from clipboard when Cut/Copy  operation is done to be pasted    somewhere.

Find/Replace : The data can be searched through the sheet. It can also be replaced.   The particular data can be found

DeleteSheet : The unwanted sheets can be deleted using this option. Currently selected sheet will   get  deleted.

Count  : This function will return the no: of selected rows/columns

Maximum  :To find out the Maximum value of selected Rows/Columns

Minimum  :To find out the Minimum value of selected Rows/Columns

Average  :This function will return the average value of selected Rows/Columns

                                                                                        Insert

Sheet : More sheets can be inserted through this menu item. Upto 14 sheets can be inserted.

InsertRow(s) : Row(s) can be inserted between the existing datas.

InsertColumn(s) :Column(s)† can be inserted between the existing datas.

DeleteRow(s) : Unnecessary rows can be deleted by selecting those rows.

DeleteColumn(s) : Unnecessary columns can be deleted by selecting those columns.

                                                                                         Chart

Pie Chart :  Pie charts can be drawn for selected data. X and Y axis value and title of the charts can      be inserted.

Bar Chart (Horizontal) : Rectangular bar charts can be drawn for selected data. X and Y axis value and title of the chart can be inserted. The shape of the chart is in horizontal rectangular form.(Horizontally drawn)

Bar Chart (Vertical) : Rectangular bar charts can be drawn for selected data. X and Y axis value† and title of the chart can be inserted. The shape of the chart is in vertical rectangular form. ( Vertically drawn)

This menu appears when right clicking anywhere within the sheets of the workbook. It has so many items. There are cut, copy, paste, ClearContent, SelectAll, DeselectAll, InsertRow, InsertColumn, DeleteRow and DeleteColumn.

The cut, copy and paste items are available in the Tools menu.

InsertRow, InsertColumn, DeleteRow and DeleteColumn are available in the Insert menu.

SelectAll :† This menu item will select all the rows and columns of the worksheet.
